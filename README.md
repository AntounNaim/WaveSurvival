## Controls
- WASD - Movement
- Space - Jump
- Mouse - Look around
- Left Click - Shoot
- T - Switch weapons
- ESC - Pause menu

## Features
- 7 enemy types with unique behaviors: Basic, Fast, Shooter, Explosive, Tank, Summoner, Minions
- 9 shop upgrades: Ammo-Cap+, Lifesteal, Critical+, DamageIncrease, ExplosiveRounds, Heal, HealthRegen+, MaxHealth+, Movement+
- Wave system with special waves (5,10,15,20,etc.)
- Loot drops from enemies
- Persistent high score system
- Enemy spawner based off of distance from player

##Bugs
-I tried my best to look for and fix any bugs I could find. To my knowledge there should be no game breaking bugs.

##Achievements & Shortcomings
-This project was really challenging in the sense that I wanted to implement as many extra things as I could in as little time as I could. To be able to present something I am actually proud of in only 2-3 days of work made me regret not having given it more time to go more in depth.
